# To-Do List App

A modern To-Do List application built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Live task counters
- Filter tasks by:
  - All
  - Active
  - Completed
- Dynamic tab highlighting
- Responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap 5

## Learning Outcomes

This project helped me practice:
- DOM Manipulation
- Event Handling
- Dynamic UI Updates
- JavaScript Functions
- CSS Styling
